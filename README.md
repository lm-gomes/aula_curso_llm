# 📚 Aula do Curso LLM 🚀

## 📖 O que é MCP? 🤔

MCP (Model‑Centric Programming) é uma abordagem de desenvolvimento que coloca **o modelo de linguagem grande (LLM)** no centro do fluxo de trabalho. Em vez de escrever código manualmente, você descreve o que deseja que o modelo faça e deixa o LLM gerar, refinar e testar o código por você. Essa técnica permite:

- **Produtividade**: escreva menos linhas de código e reduza o tempo de desenvolvimento.
- **Qualidade**: o LLM pode gerar código mais limpo e seguir padrões de estilo automaticamente.
- **Colaboração**: desenvolvedores e LLM trabalham juntos, cada um focando em suas forças.
- **Experimentação rápida**: teste diferentes abordagens sem a necessidade de reescrever tudo.

### 🚀 Como funciona?

1. **Defina a tarefa**: Escreva um prompt claro descrevendo a funcionalidade desejada.
2. **Gere o código**: O LLM responde com um esboço de código.
3. **Revise e refine**: Ajuste o prompt ou faça correções manuais se necessário.
4. **Teste**: Execute testes automáticos para garantir que o código atende aos requisitos.
5. **Itere**: Repita o ciclo até que a solução esteja perfeita.

### 🎯 Benefícios principais

| Benefício | Por quê? |
|-----------|----------|
| **Velocidade** | Menos linhas escritas = mais tempo para pensar em lógica complexa. |
| **Consistência** | O LLM segue padrões de codificação automaticamente. |
| **Aprendizado** | Desenvolvedores aprendem novas APIs e paradigmas com exemplos gerados. |
| **Escalabilidade** | Projetos grandes podem ser divididos em prompts menores e mais gerenciáveis. |

## 🛠️ Como começar?

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/lm-gomes/aula_curso_llm.git
   ```
2. **Instale as dependências** (se houver):
   ```bash
   pip install -r requirements.txt
   ```
3. **Execute os exemplos**:
   ```bash
   python example.py
   ```

### 📦 Dependências
- Python 3.9+ (ou outra linguagem conforme o exemplo).
- Biblioteca `openai` para chamadas de API.
